## Meeting Summary / Roadmap

### General Decisions
- Roles: No changes; everyone keeps their current roles.
- Game Concept: 2.5D dystopian laboratory theme is confirmed.
- Workflow: First 7 weeks are **individual** work (each member commits to their own GitHub repo). In Week 8, the team **merges** everything to produce the demo.
- Tool: Unity (final and confirmed).
- Task of the Week: Everyone should create a GitHub account and learn the basics (repository creation, commit, push).

---

## Weekly Progress Plan (Individual Work)

### **Week 1: Environment Setup & Introduction**
- Topics: Editor basics, Scenes, Prefabs, 2.5D camera setup.
- Goal: Build a simple scene/parkour where the character can move only on the X-axis with basic walls.

### **Week 2: Movement System**
- Topics: Rigidbody2D, Colliders, Jumping, Ground Check.
- Goal: A character that can move left–right and jump with a proper platformer feel.

### **Week 3: Physics & Basic Enemy**
- Topics: Patrolling enemy AI (left–right), Health system, Damage handling.
- Goal: A system where colliding with the enemy reduces health or knocks the player back.

### **Week 4: UI & Menus**
- Topics: Main menu, Pause menu, Health bar, Collectibles.
- Goal: A game loop where you start from a menu, collect coins, and can pause with "ESC".

### **Week 5: Animations**
- Topics: Idle, Walk, Jump animations, Blend Trees, transitions.
- Goal: Character animations that change based on movement (jumping, running, etc.).

### **Week 6: Stealth Mechanics**
- Topics: Enemy field of view (FOV), Chasing when seen, Returning to patrol when player hides.
- Goal: Enemies chase the player on sight and stop when the player is out of view.

### **Week 7: Puzzle & Interaction**
- Topics: Box pushing/pulling, Buttons that open doors.
- Goal: A puzzle room where pushing a box onto a button opens a door.

---

## **Week 8: FINAL (Team Integration)**
- Individual work ends; everyone switches back to their team roles.
- We combine all systems and build a 10–20 minute playable **Vertical Slice** demo.
